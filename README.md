# Дыши Носом

Privacy-first PWA-дневник практики метода К. П. Бутейко.

Приложение помогает вести локальный дневник практики: контрольная пауза, упражнения, самочувствие, сон, привычки дня, динамика и экспорт записей.

## Статус

MVP. Проект работает как статический сайт без сборки и бэкенда.

- Основное приложение: `docs/app.html`
- Лендинг: `docs/index.html`
- PWA-манифест: `docs/manifest.json`
- Service Worker: `docs/sw.js`
- Публикация: GitHub Pages из папки `docs/`

## Приватность

Данные дневника хранятся только в браузере пользователя через `localStorage`.

В текущей версии нет:

- аккаунтов;
- серверного хранения дневника;
- cookies;
- аналитики;
- рекламных или трекинговых скриптов.

Ключ основного хранилища: `dyshinosom_diary_v1`.

## Локальный запуск

```bash
cd docs
python3 -m http.server 8080
```

После запуска открыть:

```text
http://localhost:8080/app.html
```

## Структура

```text
docs/
  app.html          # основной дневник
  index.html        # главная страница
  about.html        # страница о методе
  privacy.html      # политика конфиденциальности
  terms.html        # условия и дисклеймер
  manifest.json     # PWA manifest
  sw.js             # offline/cache
  assets/           # иконки, шрифты, иллюстрации курса
```

## Перед релизом

Минимальный чек-лист:

- проверить первый вход после `localStorage.clear()`;
- создать тестовую запись и убедиться, что она появляется в журнале;
- проверить экспорт CSV/JSON/Markdown;
- проверить светлую и тёмную тему;
- проверить мобильный viewport 375px;
- проверить PWA-установку на реальных iOS/Android устройствах;
- убедиться, что в Network нет сторонних запросов.

## Медицинский дисклеймер

Проект не является медицинским изделием и не заменяет консультацию врача. Пользователь должен читать условия и противопоказания перед практикой.
